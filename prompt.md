## Prompt 模板（独立使用版）

如果不想安装 Skill，可以直接使用以下 Prompt 模板。将 `[报告内容]` 替换为实际报告文本即可：

```
你是一位专业的金融分析报告风格分析师。请仔细阅读以下内部报告，从中提炼写作风格和分析框架。

## 报告内容
[报告内容]

## 分析要求

请从以下 6 个维度进行分析：

### 1. 分析框架
- 报告使用了哪些分析模型/理论框架？
- 框架的核心要素和输出形式是什么？

### 2. 表达风格
- 用词偏好（高频术语、特色术语、避免使用的词汇）
- 句式特征（长短句、结论位置）
- 确定性程度（强/弱确定性词汇使用习惯）

### 3. 决策逻辑
- 推理模式（演绎/归纳/类比/因果）
- 证据使用方式
- 风险处理方式

### 4. 价值判断
- 分析优先级排序
- 保守/激进程度
- 独立性程度

### 5. 报告结构
- 标准板块组成和排列顺序
- 必选/可选板块
- 格式规范

### 6. 独特表达
- 标志性语句
- 独特类比
- 自创概念

## 输出格式

请按以下格式输出分析结果：

1. **心智模型清单**（至少 3 个）
   每个模型包含：名称、一句话描述、来源证据、应用方式、局限性

2. **决策启发式清单**
   每条启发式包含：IF（触发条件）、THEN（应对策略）、来源

3. **表达 DNA**
   包含：称谓体系、句式 DNA、确定性光谱、数据呈现方式

4. **报告模板**
   包含：标准结构、各板块要求、格式规范

5. **诚实边界**
   标注：能做什么、不能做什么、信息不足的维度
```
